# Internal Assessment Reference: NAPTA Platform

## What Is NAPTA?

NAPTA is a **secure data integration solution** that consolidates employee and financial data from multiple sources (e.g. SharePoint, external systems, MOM) into a central platform for **HR, Finance, Audit, and BI teams**.

***

## How It Works (Simplified)

### Step 1: Data Collection (Inbound)

*   Data is received securely from multiple systems
*   All communication is encrypted

### Step 2: Data Processing

*   Data is classified and separated by sensitivity
*   Enforced separation between Development and Production environments

### Step 3: Data Storage

Single secure database per environment, structured by sensitivity:

*   🟢 **INTERNAL** — General company data (BI access)
*   🟡 **CONFIDENTIAL** — Sensitive employee data (HR & Audit)
*   🔴 **FINANCIAL** — Payroll, budgets (Finance only)
*   ⚫ **AUDIT** — Logs and compliance evidence (Audit only)

### Step 4: Access Control

*   Entra ID SSO and role‑based access
*   Least‑privilege enforced
*   All access logged

### Step 5: Reporting & Analytics

*   Department‑specific reports
*   No cross‑visibility of sensitive datasets

### Step 6: Monitoring & Alerts

*   Continuous monitoring
*   Real‑time alerts for suspicious activity
*   Full audit trail retained
